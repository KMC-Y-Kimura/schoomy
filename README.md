# schoomy

基礎プログラミングⅡの授業で、SchooMyを使って作成したプログラムをGitHubで管理するためのリポジトリである。
SchooMy上で作成したC言語のコードを保存し、振り返り、共有することができるようにしている。

## 動作内容
このプロジェクトでは、SchooMyで作成したプログラムを動かす。
主な内容は以下の通りである。

- センサーや入力の値を読み取る
- 読み取った値に応じて処理を分ける
- 条件に応じてLEDや音などの出力を変える
- スクーミー上でプログラムの動作を確認する

※ 実際の動作内容は、作成したプログラムに合わせて変更する。
