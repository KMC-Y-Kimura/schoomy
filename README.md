# schoomy

基礎プログラミングⅡの授業で、SchooMyを使って作成したプログラムをGitHubで管理するためのリポジトリである。

このリポジトリでは、SchooMy上で作成したC言語のコードを保存し、動作内容を振り返り、共有できるようにしている。

## プログラム概要

このプログラムは、SchooMyに接続されたセンサーを使って、周囲の環境を測定し、睡眠環境の快適さを簡易的に数値化するプログラムである。

取得する値は以下の3つである。

- 温度
- 音の大きさ
- 明るさ

これらの値をもとに、それぞれの環境要素をスコア化し、合計値として「睡眠快適指数」を求める。

## 動作内容

プログラムを実行すると、温度センサー、音センサー、明るさセンサーから値を取得する。

取得した値は、それぞれ以下のように処理される。

- 温度は、23℃との差をもとにスコア化する
- 音は、アナログ入力値をもとにスコア化する
- 明るさは、明るさ指数をもとにスコア化する
- 3つのスコアを合計し、睡眠快適指数として表示する

測定結果は、シリアルモニタに表示される。

表示される内容は以下の通りである。

```text
温度
音
明るさ指数
睡眠快適指数
